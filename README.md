# Sales & Profit Dashboard (2025, USD) — Adura Datahub

## What this project is about
This project analyzes **sales performance and profitability** to answer:
- How did **net revenue** and **net profit** trend month-to-month in 2025?
- Which **regions, channels, categories, and products** drive the most profit?
- Where is margin leaking through **discounts, returns, fees, and fulfillment costs**?
- What actions can improve profitability next quarter?

## Key recommendations
1) Control discounts: target high-margin, low-return products.  
2) Reduce returns: identify top-return SKUs and improve QA + product descriptions.  
3) Optimize fulfillment: track cost per order and set weekly thresholds.
